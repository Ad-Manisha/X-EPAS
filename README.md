# X-Employee Performance Assessment System (X-EPAS)

**X-Employee Performance Assessment System (X-EPAS)** is an AI-driven web application designed to provide **transparent, fair, and evidence-based evaluations** of employee contributions. The system tracks employee tasks, submissions, and attendance, and uses objective metrics to generate performance scores, minimizing personal bias in performance reviews.

## Key Features
- Task assignment and management for employees
- Submission tracking with GitHub integration
- AI-driven evaluation of task submissions
- Attendance monitoring for punctuality and consistency
- Clear separation of entities for scalability and maintainability

## Tech Stack
- **Frontend:** React  
- **Backend:** FastAPI  
- **Database:** MongoDB  
- **Data Processing & AI:** Python  

## Objective
To create a fair, transparent, and data-driven system that supports performance assessment, feedback, and potential promotion decisions.
